What Makes a Website


Author: Cesar Mejia | Date: 05/11/19

HTML, CSS, and JavaScript are use to create any website. HTML is used to set the structure of any website. whether be it setting the main heading, it's paragraphs or even tables. However, HTML alone would make a very bare website, just black text with some images. Thus, including CSS into the mix gives a website a more appealing look. Adding colors to backgrounds, to text and even allowing us to modify images, CSS allows us to dress a website and make it look nice and professional. However, if we want the cool effects or want to add interactivity to our website we would add JavaScript.
